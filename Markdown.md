# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочкам (*). Например, *вот так*.

Чтобы выделить текст жирным, надо обрамить его двойными звездочками (**). Например, **вот так**.
